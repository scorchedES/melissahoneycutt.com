# melissahoneycutt.com

Static website for Melissa Honeycutt, focused on AI implementation for real estate agents.

## Site Structure

This is a static HTML/CSS site hosted on GitHub Pages.

- Main entry point: `index.html`
- Shared styles: `styles.css`
- Image assets: `assets/images/`
- Core pages: `services.html`, `use-cases.html`, `about.html`, `faq.html`, `contact.html`
- Not-found page: `404.html`

## Deployment Notes

GitHub Pages serves this site from the repository root. The custom domain is configured with `CNAME` as `melissahoneycutt.com`.

Domain DNS is managed separately through GoDaddy. Keep the GitHub Pages custom domain and GoDaddy DNS records aligned before launch.
